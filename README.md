## 📖 概述
**一句话介绍**：  
*“这是一个以酿酒、调酒、酒馆经营为核心的 Minecraft 模组包，包含丰富的酿造系统、独特的酒类物品和社交玩法！”*

**核心特色**：

- ✅ **酒馆与社交**：开设酒馆，邀请好友举杯共饮。
- ✅ **兼容性**：支持主流 Mod 和版本（如 1.20.1）。
